Это проект по сбору и классификации иероглифов [Теневиля](https://ru.wikipedia.org/wiki/%D0%A2%D0%B5%D0%BD%D0%B5%D0%B2%D0%B8%D0%BB%D1%8C).
В настоящий момент работа ведется на русском языке.

This project is aimed at the collection and classification of [Tenevil's](https://en.wikipedia.org/wiki/Tenevil) characters.
At the moment the work is conducted in Russian. An English translation is possible in the future.
